🔥 كيفية ربط Firebase
الخطوة 1 — إنشاء مشروع Firebase
  1. اذهب إلى console.firebase.google.com
  2. اضغط Create a project → أدخل اسم المشروع
  3. من القائمة الجانبية: Build → Authentication → Get started → Email/Password → Enable
  4. من القائمة الجانبية: Build → Realtime Database → Create database → Start in test mode
الخطوة 2 — إضافة المستخدم
  1. من Authentication → Users → Add user
  2. أدخل: [email protected]
  3. كلمة المرور: Jamx@1996
الخطوة 3 — نسخ الإعدادات في الكود
  1. Project Settings (⚙) → Your apps → Add app → Web
  2. انسخ firebaseConfig وضعه في ملف HTML بدلاً من القيم الوهمية
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "myproject.firebaseapp.com",
  databaseURL: "https://myproject-rtdb.firebaseio.com",
  projectId: "myproject",
  ...
};
